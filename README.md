# .github

Community health files and default templates for all [@mrlesmithjr](https://github.com/mrlesmithjr) repositories.

These files apply globally across all repos unless overridden at the individual repo level.

## Contents

| File | Purpose |
|------|---------|
| `CONTRIBUTING.md` | Contribution guidelines |
| `CODE_OF_CONDUCT.md` | Community standards |
| `FUNDING.yml` | GitHub Sponsors configuration |
| `SECURITY.md` | Vulnerability reporting process |
| `.github/ISSUE_TEMPLATE/` | Bug report and feature request templates |
| `.github/PULL_REQUEST_TEMPLATE.md` | PR checklist |
| `stale.yml` | Stale issue/PR automation |
